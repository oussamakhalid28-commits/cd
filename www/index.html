CD App — بداية تطبيق Android

تم تهيئة المشروع ليستخدم مجلد www/ كـ webDir لـ Capacitor.

لتجربة التطبيق على جهاز Android:

  1. npm install
  2. npx cap add android
  3. npx cap sync
  4. npx cap run android

ملاحظة للمطورين

استبدل هذا الملف أو أنشئ بنية مشروعك (React/Vue/Ionic) ثم تأكد أن عملية البناء تخرج الملفات إلى www/.

HTML: www/index.html
Capacitor config: capacitor.config.json
Scripts: npm run build  |  npm run prepare  |  npm run android
